Dog Adoption Website mockup Prototype
How to run
1.Download mockup-proto from our repository
2.Open VSCode
3.Configure .env to match your MySQL profile
4.Open dog_adoption_database.sql through MySQL Workbench
5.Run the script
6.Configure mySQL schema to add dog_adoption_database entry by navigate Server--> User and Priviledge --> choose your schema --> add entry dog_adoption_database
7.Back to VSCode and open terminal
8.cd back-end-info --> cd back-end
9.type "npm i"
10.after installation, type "npm start"
11.enjoy our prototype!
